# Advanced Guides Overview

This directory contains documentation for advanced development scenarios and customization within the Internara project. These guides are intended for developers who need to extend the core functionality of the modular architecture or customize the development workflow.

---

## Guides in this Section

### 1. [Custom Module Generator](custom-module-generator.md)
Explains how to customize the generation of new modules, including the use of custom stubs and the "Base Module" cloning pattern for highly specific module structures.


---

## When to use these guides
Refer to these documents only after you have a solid understanding of the core architecture and standard workflows defined in the [Master Documentation](../main-documentation-overview.md). Advanced techniques should be used sparingly and only when the default conventions are insufficient for a specific business requirement.
