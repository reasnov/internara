# Auth Module

The `Auth` module manages all aspects of user authentication, including login, registration, and email verification. it provides the entry point for users to securely access the Internara platform.

## Purpose

-   **Access Control:** Verifies user identity before granting access to protected resources.
-   **Onboarding:** Handles new user registration workflows.
-   **Security:** Enforces session management and email verification policies.

## Key Features

-   **Livewire Authentication:** Reactive login and registration forms.
-   **Role-Based Redirection:** Automatically routes users to their relevant dashboard upon login.
-   **Email Verification:** Integration with Laravel's verification system to ensure valid accounts.
