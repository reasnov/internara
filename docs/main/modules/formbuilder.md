# FormBuilder Module

The `FormBuilder` module enables dynamic form creation and rendering. it is primarily used for
generating flexible assessment forms and journal templates.

## Purpose

- **Flexibility:** Allows administrators to define custom form fields without code changes.
- **Data Collection:** Standardizes how dynamic data is captured and stored.
- **Extensibility:** Supports various field types (Text, Radio, Checkbox, etc.).

## Key Features

- **Dynamic Renderer**: Livewire components that render forms based on database definitions.
