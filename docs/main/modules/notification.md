# Notification Module

The `Notification` module provides a unified system for sending alerts and messages to users through various channels (Database, Mail, etc.).

## Purpose

-   **Communication:** Keeps users informed about important events.
-   **Centralization:** Standardizes the creation and dispatching of system notifications.
-   **Extensibility:** Easily supports multiple delivery channels.

## Key Features

-   **Modular Notifications**: Pre-defined notification classes for common lifecycle events (e.g., account activation, internship approval).
