# School Module

The `School` module manages the identity and configuration of the educational institution.

## Purpose

- **Institutional Profile:** Stores school details like name, address, and logo.
- **Configuration:** Manages school-specific settings.

## Key Features

- **SchoolService**: Logic for managing the primary school record.
- **Media Integration**: Specialized handling for the school's logo.
