# UI Module

The `UI` module is the project's internal design system. it provides global styles, shared layouts, and reusable Blade components.

## Purpose

-   **Consistency:** Maintains a unified visual identity across all modules.
-   **Reusability:** Provides a library of battle-tested components.
-   **Centralization:** Acts as the single source of truth for global layouts and CSS.

## Key Features

-   **[UI/UX Design System](../ui/ui-overview.md)**: Standardized components and principles.
-   **Global Layouts**: Base Blade layouts for authenticated and guest users.
-   **Slot Registry**: A system for cross-module UI injection.
