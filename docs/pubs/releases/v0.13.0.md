## üõ† What's Changed?

### Added

- **Metadata SSoT**: Introduced `MetadataService` in the `Core` module as the authoritative source
  for application identity and author verification.
- **Localization Service**: Centralized locale management in the `UI` module, ensuring strict
  adherence to the "Thin Component" mandate for the Language Switcher.
- **New UI Components**: Introduced `x-ui::stat` for standardized metric visualization and enhanced
  `x-ui::avatar` with better placeholder support.
- **Mandatory Quality Gate**: Integrated `composer test` as a standard verification step in the
  engineering lifecycle, documented in the Testing Guide.
- **Universal Agent Workflows**: Formalized and documented 10 engineering lifecycles in the
  `docs/agents/workflows/` directory to standardize AI-assisted development.
- **Browser Stability Layer**: Added global browser arguments (`--no-sandbox`, etc.) to the testing
  baseline to ensure stability in Linux and virtualized environments.

### Changed

- **Architectural Re-alignment**: Reconstructed all **Application Blueprints** (ARC01 series) to
  reflect the complete project roadmap from Genesis to current.
- **Directory Standardization**: Migrated all remaining `Traits` directories to `Concerns` and
  relocated technical utilities to `Support` or `Functions` according to new conventions.
- **Enhanced Testing CLI**: Refactored `app:test` to support sequential sub-segment execution and
  provide high-fidelity summary tables with duration tracking.
- **Service Layer Optimization**: Refactored domain services (`UserService`, `OnboardingService`,
  `ProfileService`, etc.) to improve isolation and utilize standardized `EloquentQuery` patterns.
- **Refined Exception Handling**: Updated `HandlesAppException` to utilize localized keys and
  enforced secure error abstraction for production environments.

### Fixed

- **Illegal Instruction (SIGILL) Resilience**: Implemented `try-catch` blocks and process signaling
  protection in the test orchestrator to handle CPU/Sandbox incompatibilities gracefully.
- **Pest Framework Scoping**: Resolved an issue in `tests/Pest.php` where modular tests were not
  correctly extending the base `TestCase` due to restrictive glob patterns.
- **Normalization Logic**: Fixed redundant slash handling in `Formatter` for both file system paths
  and PHP namespaces.
- **Onboarding Integrity**: Corrected `OnboardingService` to interact with domain models exclusively
  via Service Contracts, preserving modular isolation.
