## ðŸ›  What's Changed?

### Added

- **PHPFlasher**: Added `php-flasher/flasher-laravel` as the primary notification provider.
- **Theme Sync**: Implemented a theme watcher to ensure PHPFlasher's appearance matches the current
  system theme (Light/Dark).

### Changed

- **Notification Helper**: Refactored the `notify()` helper in the `Notification` module to act as a
  wrapper for `flash()`, maintaining backward compatibility while leveraging the new engine.
- **Livewire Integration**: Updated all Livewire components to use `flash()` directly for feedback,
  improving code clarity and maintainability.
- **Component Props**: Removed the unused `aos` property from all UI components.

### Fixed

- **Attendance Feedback**: Fixed an issue where attendance recording feedback was not reliably
  displayed due to legacy event-based notification dispatching.
- **Slot Manager Stability**: Improved the stability of Livewire components rendered through the
  `SlotManager` by ensuring consistent, MD5-hashed stable IDs.

### Removed

- **AOS Library**: Removed `aos` dependency from `package.json` and CSS/JS entry points.
- **Legacy Toast Component**: Deleted `modules/UI/resources/views/components/toast.blade.php`.
- **Notification Bridge**: Deleted `modules/UI/src/Livewire/NotificationBridge.php`.
