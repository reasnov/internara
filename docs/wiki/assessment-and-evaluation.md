# Assessment and Evaluation

As the internship period ends, the system aggregates all data for final evaluation and
certification.

---

## 1. Compliance Scoring
- **Auto-Calculation**: Internara automatically calculates the **Participation Score** based on
  attendance and approved journals.

## 2. Qualitative Evaluation
- **Action**: Both Instructors and Mentors submit final performance rubrics for each student.

## 3. Final Certification
- **Navigation**: `Assessment` -> `Final Grades`
- **Action**: Review the composite score and generate the **Competency Achievement Report** (PDF).
