# Program Preparation: Cycles and Partners

Before students can register, the administrator must define the parameters of the internship
period.

---

## 1. Defining Internship Programs
- **Navigation**: `Internship` -> `Programs`
- **Action**: Create a new program, defining the Start Date, End Date, and Target Academic Year.
- **Why?**: This acts as the container for student registrations and compliance tracking.

## 2. Industrial Partner Registry
- **Navigation**: `Internship` -> `Companies`
- **Action**: Register the industry partners (companies) that will host students.
- **Why?**: Companies are required to create specific placement slots.

## 3. Placement Availability
- **Navigation**: `Internship` -> `Placements`
- **Action**: Create available slots for each company, specifying which department the slot is for.
